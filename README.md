# Setup

1. Install dependencies

```
cd eventgo-frontend
npm i
```

2. Populate `.env.local`

-   Copy the content of `eventgo-frontend/.env.example` and rename it to `eventgo-frontend/.env.local`.

## Running Frontend

```
cd frontend
npm run dev
```

## Running Backend Microservices

```
cd eventgo-backend

docker compose up
```
